# ChatList

Минимальная программа на Python с графическим интерфейсом на PyQt5.

## Установка

1. Убедитесь, что у вас установлен Python 3.8 или выше
2. Установите зависимости:

```powershell
pip install -r requirements.txt
```

## Запуск

### Вариант 1: Запуск из исходного кода

```powershell
python main.py
```

### Вариант 2: Создание исполняемого файла .exe

1. Установите PyInstaller (если ещё не установлен):
```powershell
pip install pyinstaller
```

2. Запустите скрипт сборки:
```powershell
.\build.ps1
```

3. Исполняемый файл будет создан в папке `dist\ChatListApp.exe`

4. Запустите приложение:
```powershell
.\dist\ChatListApp.exe
```

## Описание

Это простое приложение демонстрирует базовые возможности PyQt5:
- Создание главного окна
- Добавление виджетов (метка и кнопка)
- Обработка событий нажатия кнопки
- Обновление текста в реальном времени

При нажатии на кнопку счётчик увеличивается и обновляется текст на экране.
