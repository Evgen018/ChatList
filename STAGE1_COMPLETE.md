## 💡 Рекомендации

### Перед началом работы:

1. **Создайте .env файл:**
```powershell
Copy-Item .env.example .env
```

2. **Добавьте свои API-ключи в .env:**
```env
OPENAI_API_KEY=sk-proj-ваш_ключ
DEEPSEEK_API_KEY=ваш_ключ
GROQ_API_KEY=ваш_ключ
```

3. **Инициализируйте базу данных:**
```powershell
python -c "from db import Database; db = Database(); db.init_schema(); print('БД готова'); db.close()"
```

4. **Запустите тесты для проверки:**
```powershell
python test_modules.py
```
