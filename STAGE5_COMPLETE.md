## 🚀 Запуск приложения

### Первый раз:

```powershell
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Создать .env файл
Copy-Item .env.example .env

# 3. Добавить API-ключи в .env (откройте в редакторе)

# 4. Инициализировать БД
python -c "from db import Database; db = Database(); db.init_schema(); db.close()"

# 5. Запустить
python main.py
```

### Повторный запуск:

```powershell
python main.py
```

### Создание .exe файла:

```powershell
.\build.ps1
```

Результат: `dist\ChatListApp.exe`

## 🎨 Особенности интерфейса

### Цветовая схема

- 🟢 **Зелёная** кнопка "Отправить" - основное действие
- 🔵 **Синяя** кнопка "Сохранить" - важное действие
- ⚪ **Серые** кнопки - второстепенные действия

### Индикаторы состояния

- ✅ **Зелёная галочка** - всё настроено
- ⚠️ **Жёлтый треугольник** - требуется внимание
- ❌ **Красный крестик** - ошибка

### Статус бар

Отображает:
- "Готов к работе" - в ожидании
- "Отправка запроса..." - процесс выполнения
- "Получено X из Y ответов" - результат
- "Сохранено N результат(ов)" - после сохранения
